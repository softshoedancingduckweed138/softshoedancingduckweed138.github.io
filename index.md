## 🎯 About This Tool

Zahori finds the hidden video source behind web pages. Many websites protect their video files to prevent direct downloads. This software acts like a web browser to locate those hidden files for you. It supports common streaming formats like HLS, DASH, and MP4. It also remembers how specific websites work, which makes future downloads faster. You use this tool to save video files from these streams to your Windows machine for offline viewing.

## 🚀 How to Use Zahori

After you install the software, it opens as a small window. You don't need to write code to make it work.

1.  Copy the web address (URL) of the page containing the video.
2.  Paste that address into the top bar of the application window.
3.  Click the "Find" button.
4.  Wait a few seconds while the software scans the page.
5.  When the software finds a stream, it displays a list of video resolutions and sizes.
6.  Select the resolution you prefer.
7.  Click the "Download" button to save the video file to your folder.

## 🔍 Troubleshooting Common Issues

Sometimes, websites change their design. If the software fails to find a video, try these steps:

*   Refresh the web page in your regular browser.
*   Ensure that you copied the full web address with the "https://" part.
*   Close the software and reopen it.

If the software still fails, the specific website might block automated users. You can report these issues through the GitHub repository link provided above.
